## Summary

`v2.0.0` is planned as the release that reduces AOF documentation to a small canonical set and makes installer/bootstrap the main entrypoint for managed projects.

The release surface centers on:

- canonical doc reduction
- `aof init` bootstrap
- `aof upgrade` migration
- explicit managed-project default
- a concrete acquisition flow from GitHub release/tag to local install

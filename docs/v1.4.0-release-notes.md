## Summary

`v1.4.0` finalizes the Human Visibility Layer contract that lets AOF runtime evidence be read as `status / timeline / flow` without requiring a rich viewer implementation.

The release centers on:

- runtime event / log contract
- human visibility output formats
- static vs runtime-used distinction
- status / timeline / flow views

## Included In v1.4.0

Release scope is defined in [docs/v1.4-release-definition.md](./v1.4-release-definition.md).

## Explicit Simplifications From v1.3

`v1.4` does not treat the following as core release gates:

- rich GUI viewer implementation
- hosted observability service
- analytics overlays
- automated explanation generation

These remain possible later expansions, but the main target is the visibility contract itself.

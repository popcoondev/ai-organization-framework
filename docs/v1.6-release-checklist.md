## Blocking Conditions

次のどれかが残る場合、`v1.6 ready` とは言わない。

1. experience-validation が phase ではなく単なる advice に留まる
2. `North Star Goal` と `Next Value Slice` の関係が未定義
3. Experience Steward の責務が曖昧
4. viewer fidelity と editor fidelity が未分離
5. `expectation-mismatch` が first-class signal として定義されていない
6. proof category taxonomy が無く、spec completeness と product-feel truth が混同される

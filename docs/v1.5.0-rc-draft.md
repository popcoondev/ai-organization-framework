## Summary

- `v1.5` is currently defined as a trust-assumptions release, not a runtime-expansion release
- the release target is to make AOF's reliability boundaries explicit before further automation
- current repo state now has the trust-assumption companion docs assembled

## Scope Of This Release

### Included

- routing capability assumptions
- council independence and execution-pattern limits
- clarification / discovery quality floor
- human review cadence by usage level
- structured-output fallback limits
- context completeness guidance

### Explicitly Not Core In v1.5

- stronger model-routing automation
- full multi-model orchestration
- probabilistic calibration system
- rich reviewer staffing libraries
- hosted governance assurance service

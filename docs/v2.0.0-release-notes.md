# AOF v2.0.0 Release Notes Draft

- version: `v2.0.0`
- status: ready
- release note revision commit SHA: `29b8c2a`

## Summary

`v2.0.0` is planned as the release that turns AOF into a smaller, installable system:

- fewer canonical docs
- an explicit managed-project default
- a practical acquisition path
- an installer/bootstrap entrypoint via `aof init`
- a migration path via `aof upgrade`

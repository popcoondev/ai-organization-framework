## Summary

- `v1.2` is currently defined as a concept-layer release, not a runtime-surface expansion
- the release target is to make discovery-first intake, lightweight organization sizing, and interpretation lenses explicit
- current repo state has the release definition, but not yet the full evidence set needed to claim readiness

## Scope Of This Release

### Included

- `Discovery Layer`
- `Knowledge State`
- lightweight `Organization Pattern`
- `scope sizing -> initial organization selection`
- `Interpretation Lenses`

### Not Newly Claimed

- advanced analytics dashboards
- safe multi-writer concurrency
- hosted discovery workspace
- full Policy DSL
- large actor taxonomy

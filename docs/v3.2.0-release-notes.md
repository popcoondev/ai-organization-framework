# AOF v3.2.0 Release Notes

- target version: `v3.2.0`
- current package version: `3.2.0`
- status: in progress
- release note revision commit SHA: `to be fixed by the release tag`

## Summary

`v3.2.0` is the release that makes Need Validation a first-class pre-project runtime gate.

Before this release, AOF was strongest after a project already existed.  
`v3.2.0` extends that contract so the runtime can now say:

- the stated need is too weak
- the stated need is misframed
- more evidence is required
- an experiment should happen first
- no project should be created

The release claim is not "Discovery is complete."  
The claim is "project creation no longer bypasses need validation."

## Highlights

### 1. Need Validation Is Now A Canonical Artifact Family

`v3.2.0` adds:

- problem statement
- value hypothesis
- alternative analysis
- experiment proposal
- project charter
- need validation record

as schema-backed CLI artifacts rather than prose-only guidance.

### 2. Runtime Planning Is Now Gated

clarification completion no longer advances directly into planning.

The runtime now:

- stops at `need-validation`
- emits a need validation decision
- rejects downstream council execution until the gate is advanced

### 3. Planning Requires Explicit Approval Evidence

`need-validation-advance` promotes a session into `planning` only when:

- the need validation record is approvable
- the project charter is linked
- project creation is explicitly recommended

This makes the planning transition itself auditable.

### 4. Need Validation Has A Benchmark Surface

`need-validation-benchmark` evaluates the initial benchmark family:

- `NV-001 Need Validation`
- `NV-002 False Problem Rejection`
- `NV-003 Value Hypothesis Quality`
- `NV-004 Alternative Discovery`
- `NV-005 Experiment Efficiency`
- `NV-006 Project Readiness`

## Boundary

`v3.2.0` does not claim:

- full Discovery Layer completion
- autonomous business discovery
- human-review removal
- generalized market research automation

It claims a stronger pre-project governance contract.

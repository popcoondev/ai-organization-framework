## Summary

- `v1.4` is currently defined as a Human Visibility Layer release, not a viewer-product release
- the release target is to make runtime evidence readable as `status / timeline / flow`

## Scope Of This Release

### Included

- runtime event / log contract
- human visibility output formats
- static vs runtime-used distinction
- human-facing status, timeline, and flow views

### Explicitly Not Core In v1.4

- rich GUI implementation
- hosted observability service
- analytics dashboards
- automated explanation generation

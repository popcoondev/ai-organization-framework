# AOF v2.1.0 Release Notes Draft

- version: `v2.1.0`
- status: draft
- release note revision commit SHA: `80352ea`

## Summary

`v2.1.0` is planned as the release that turns AOF's installed project state into an explicit organization model.

Planned highlights:

- AOF core definition moves from agent-centered framing to organization-centered framing
- new `.aof/organization.json` bootstrap artifact
- role and agent separation
- council / team / contract / dependency / knowledge / metrics / lifecycle representation
- `aof init` and `aof upgrade` support for organization manifests

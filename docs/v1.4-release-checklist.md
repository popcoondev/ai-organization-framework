# AOF v1.4 Release Checklist

`v1.4` を release-ready と判定する前に確認する checklist。

## How To Use This Checklist

1. gate ごとに必要 evidence を確認する
2. `v1.4` では visibility contract と output formats を正本とする
3. missing evidence がある場合は `v1.4 ready` と言わない

## Blocking Conditions

次のどれかが残る場合、`v1.4 ready` とは言わない。

1. runtime event contract が product-agnostic に読めない
2. `status_card / timeline_feed / flow_snapshot` の format が定義されていない
3. static vs runtime-used distinction が human-facing output として読めない
4. `v1.3` の mission control 語彙と visibility 層が矛盾している

# AOF v1.2 Release Checklist

`v1.2` を release-ready と判定する前に確認する checklist。  
この文書は [docs/v1.2-release-definition.md](./v1.2-release-definition.md) の gate を、repo 内の evidence と concept-validation artifact に落としたものとする。

## How To Use This Checklist

1. gate ごとに必要 evidence を確認する
2. `v1.2` では concept docs と validation artifact を正本とする
3. missing evidence がある場合は `v1.2 ready` と言わない

`v1.2` の役割は runtime surface を広げることではなく、discovery-to-delivery の入口を concept layer として固定することである。

### Gate 2: Discovery Layer Is Explicit

Required evidence:

- [x] docs define the pre-Need discovery flow
- [x] AOF no longer assumes every request already has a valid `Need`

Primary evidence:

- [docs/discovery-phase.md](./discovery-phase.md)

- satisfied

### Gate 3: Knowledge State Is Minimal And Useful

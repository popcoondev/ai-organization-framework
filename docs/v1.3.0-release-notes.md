## Summary

`v1.3.0` finalizes the operating-model layer that turns AOF into a faster startup and long-running mission-control framework without narrowing it to front-AI-only use.

The release centers on:

- discovery-first acceleration
- team assembly from sizing
- explicit operating goal sizing
- autonomy budget and escalation boundary
- mission control with drift response

## Included In v1.3.0

Release scope is defined in [docs/v1.3-release-definition.md](./v1.3-release-definition.md).

## Explicit Simplifications From v1.2

`v1.3` does not treat the following as core release gates:

- interpretation-lens expansion
- cross-domain minimal-model validation by itself
- further verification/dashboard growth

These remain useful references, but the main target moves to operating speed and control.

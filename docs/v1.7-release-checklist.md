3. missing evidence がある場合は `v1.7 ready` と言わない

## Blocking Conditions

次のどれかが残る場合、`v1.7 ready` とは言わない。

1. AI Orchestrator の責務境界が曖昧
2. topology と usage level が混同される
3. parallel execution と independence claim が混同される
4. state ownership が曖昧
5. schema extension が互換性を壊す

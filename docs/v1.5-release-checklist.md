# AOF v1.5 Release Checklist

`v1.5` を release-ready と判定する前に確認する checklist。  
この文書は [docs/v1.5-release-definition.md](./v1.5-release-definition.md) の gate を、repo 内の evidence と trust-assumption docs に落としたものとする。

## How To Use This Checklist

1. gate ごとに必要 evidence を確認する
2. `v1.5` では trust assumptions と operating assumptions を正本とする
3. missing evidence がある場合は `v1.5 ready` と言わない

## Blocking Conditions

次のどれかが残る場合、`v1.5 ready` とは言わない。

1. routing trust が model capability 非依存のように読める
2. council execution pattern ごとの独立性強度が定義されていない
3. clarification / discovery quality floor が無い
4. usage level ごとの human review cadence が無い
5. structured output fallback や context completeness の限界が明記されていない

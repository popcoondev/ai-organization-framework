## Scope Of This Release

### Included

- machine-readable process telemetry
  - `stage_transitions[]`
  - `routing_mode_history[]`
  - `reopen_count`
- `outcome-report` CLI
- committed evidence session with outcome writeback
- explicit runtime usage policy
- explicit serial-only concurrency policy

### Not Newly Claimed

- advanced analytics dashboards
- cross-session comparison rollups
- branch/merge concurrency
- hosted metrics backend
- richer policy DSL formalization

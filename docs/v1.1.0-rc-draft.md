## Summary

- `v1.1` の主眼だった measurement foundation と operational policy は current candidate で揃っている
- `v1.0.0` の core runtime surface は維持されたまま、process telemetry と outcome writeback が追加されている
- `framing-only` と `runtime-on` の使い分け、および same-session serial mutation policy も docs と runtime の両方で固定された

## Scope Of This Release

### Included

- machine-readable process telemetry
  - `stage_transitions[]`
  - `routing_mode_history[]`
  - `reopen_count`
- `outcome-report` CLI
- committed evidence session with outcome writeback
- explicit runtime usage policy
- explicit serial-only concurrency policy

### Not Newly Claimed

- advanced analytics dashboards
- cross-session comparison rollups
- branch/merge concurrency
- hosted metrics backend
- richer policy DSL formalization
